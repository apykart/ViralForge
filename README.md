# ViralForge AI – Turn Videos Into Viral Content

AI-powered SaaS platform for creators.

## Features
- Upload video → 4 AI thumbnails, SEO titles, description, tags, hashtags, hooks, short captions, viral score.
- Firebase Auth + Firestore + Storage.
- Admin panel (super admin: dailyway@gmail.com).
- Premium UI with dark/glassmorphism.

## Setup
1. Clone repo.
2. Create Firebase project, enable Auth, Firestore, Storage.
3. Create `.env` with your API keys (Gemini, Replicate optional).
4. Deploy on Vercel / Render / Firebase Hosting.

## Environment Variables
VITE_FIREBASE_API_KEY=...
VITE_GEMINI_API_KEY=...
